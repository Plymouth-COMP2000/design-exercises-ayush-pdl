**Student:** Ayush Paudel  
**Module:** Software Engineering 2 (COMP2000)  
**University:** University of Plymouth  

---

##  Overview
A high-fidelity Android Studio (Java/XML) prototype for a **Restaurant Management App** built as part of COMP2000 coursework.  
The design follows **User-Centred Design** and **HCI principles**, focusing on usability, accessibility, and responsive layout.

---

##  User Roles
- **Staff:** Login, Dashboard, Manage Menu, Reservations, Notifications  
- **Guest:** Login, Make Reservation, My Reservations, Notifications  

---

##  Main Features
- Material Design layouts  
- Inline input validation (Reservation Form)  
- Colour-coded booking statuses  
- Toggle-based notification settings  
- Clean and responsive UI using ConstraintLayout and RecyclerView  

---

## Tech Stack
Android Studio • Java/XML • Material Design • RecyclerView • ConstraintLayout
